Web3 Data Integrity for the rest of us.

Blobcache is the data layer

Blobcache forms a private web-of-trust storage network.

Each Node manages a set of Volumes, which can be shared with other Node's or kept private.

Each Volume contains a single hash-linked data structure which can be transactionally edited.
